# AlchemyCMS mixins for VueJS

## Install

```
yarn add @alchemy_cms/vue
```

## Usage

### In a page component

```js
import { AlchemyPage } from "@alchemy_cms/vue/mixins"

export default {
  mixins: [AlchemyPage]
}
```

### In a element component

```js
import { AlchemyElement } from "@alchemy_cms/vue/mixins"

export default {
  mixins: [AlchemyElement]
}
```
